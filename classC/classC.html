Subnetting Class C IP addresses.

To learn more about the IP classes Click here!

Hints
The default subnet mask IP address for the class C IP addresses is 255.255.255.0
Hints
Each subnet mask is a sequence of contiguous number of 1 bits followed by some number of 0 bits in a 32-bit binary representation.
The CIDR notation is expressed as "/X," where X is the number of bits set to 1 in the subnet mask.
For example:
CIDR Notation: /27
Subnet Mask IP address: 11111111.11111111.11111111.11100000 (255.255.255.224 in decimal)
In this case, there are 27 contiguous bits set to 1, so the CIDR notation is "/27."



How to Convert ?
Hints
First, convert both the computer's IP address and subnet mask IP address to the binary forms.
Second, perform bitwise AND operation between binary representation of the computer's IP address and the binary representation of the subnet mask IP address.
Third, convert the result of the AND operation back to decimal to obtain the network IP address in a dotted-qad form.
For example:
Computer's IP address: 192.168.1.35
Subnet Mask IP address: 255.255.255.192
Computer's IP Address(192.168.1.35) in binary: 11000000.10101000.00000001.00100011
Subnet Mask IP Address(255.255.255.192) in binary: 11111111.11111111.11111111.11000000
Result of AND operation: 11000000.10101000.00000001.00000000
Network IP Address: 192.168.1.0
Hints
Convert the both computer's IP address and subnet mask IP address to their binary forms.
Invert all the bits of the subnet mask
Perform bitwise OR operation between binary representation of the computer's IP address and the binary representation of the inverted subnet mask IP address.
Convert the result of the OR operation back to decimal to obtain the broadcast IP address.
For example:
IP address: 192.168.1.48
Subnet Mask IP address: 255.255.255.192
1. Computer's IP Address (192.168.1.35) in binary: 11000000.10101000.00000001.00100011
2. Subnet Mask IP address: 11111111.11111111.11111111.11000000
3. Inverted Subnet Mask IP address in binary:00000000.00000000.00000000.00111111
4. Result of OR operation between 1 and 3 : 11000000.10101000.00000001.00111111
5. Broadcast IP address: 192.168.1.63
Hints
Simply increment the network IP address by 1.
For example:
Network IP Address: 215.125.154.128
First Host IP Address: 215.125.154.129
Hints
Simply decrement the broadcast IP address by 1.
For example:
Broadcast IP Address: 215.125.154.255
Last Host IP Address: 215.125.154.254
Hints
Number of usable host IP address will be 2(32 - CIDR number) - 2.
We subtract two for the network and broadcast IP addresses.
For example:
If CIDR number is /30
then the number of assignable IP addresses is
= 2(32 - 30) - 2 = 2.
