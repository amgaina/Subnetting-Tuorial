Subnetting Class C IP address.

Learn more about the IP-classes Click here!

How to do ??
Since it is a class C network, default subnet mask will be 255.255.255.0.
In binary, it will be 11111111.11111111.11111111.00000000.
How to do ??
Each subnet mask is a sequence of contiguous 1s followed by zeros in a 32-bit binary representation.
The CIDR notation is expressed as "/X," where X is the number of bits set to 1 in the subnet mask.
For example:
CIDR Notation: /27
Subnet Mask: 11111111.11111111.11111111.11100000 (255.255.255.224 in decimal)
In this case, there are 27 bits set to 1, so the CIDR notation is "/27."



How to Convert ?
How to do ??
Convert the both IP address and subnet mask to the binary form
Perform bitwise AND operation between binary representation of the IP address and the binary representation of the subnet mask.
Convert the result of the AND operation back to decimal to obtain the network address.
For example:
IP address: 192.168.1.35
Subnet Mask: 255.255.255.192
IP Address (192.168.1.35) in binary: 11000000.10101000.00000001.00100011
Subnet Mask (255.255.255.192) in binary: 11111111.11111111.11111111.11000000
Result of AND operation: 11000000.10101000.00000001.00000000
Network Address: 192.168.1.0
How to do ??
Convert the both IP address and subnet mask to the binary form
Invert all the bits of the subnet mask
Perform bitwise OR operation between binary representation of the IP address and the binary representation of the inverted subnet mask.
Convert the result of the OR operation back to decimal to obtain the broadcast address.
For example:
IP address: 192.168.1.48
Subnet Mask: 255.255.255.192
1. IP Address (192.168.1.35) in binary: 11000000.10101000.00000001.00100011
2. Subnet Mask: 11111111.11111111.11111111.11000000
3. Inverted Subnet Mask in binary:00000000.00000000.00000000.00111111
4. Result of OR operation between 1 and 3 : 11000000.10101000.00000001.00000000
5. Broadcast Address: 192.168.1.0
How to do ??
Simply increment the network IP address by 1.
Network IP Address: 215.125.154.128
First Host IP Address: 215.125.154.129
How to do ??
Simply decrement the broadcast IP address by 1.
Broadcast IP Address: 215.125.154.255
Last Host IP Address: 215.125.154.254
How to do ??
Number of usable host will be 232 - subnet mask length - 2.
We subtract two for the network and broadcast addresses.
For example:
If subnet mask length( CIDR) : /30
Number of usable host IP = 232 - 30 - 2 = 2.
